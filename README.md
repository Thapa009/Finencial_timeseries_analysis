📈 Financial Time Series Analysis
A hands-on repository to learn financial time series analysis from scratch.
Focuses on implementing core concepts, experimenting with real data, and building intuition through practice rather than theory.

❗ Prerequisites

If you are a beginner with no prior experience in Python, statistics, or financial concepts, it is highly recommended to first build a foundation in these areas before diving into financial time series analysis.

Helpful background knowledge includes:

Python programming
Linear algebra and probability basics
Statistics (mean, variance, distributions)
Time series fundamentals (stationarity, autocorrelation)
Basic finance concepts (returns, volatility, risk)

Having these prerequisites will make it significantly easier to understand the models and implementations provided in this repository.

📖 What You'll Learn

In this repository, I implement key financial time series methods from scratch to develop a deep understanding of their mechanics and real-world applications.

To maintain consistency and clarity, most techniques are applied to the same datasets (e.g., stock prices, commodities, forex). This makes it easier to compare how different methods behave and interpret results effectively.

In the mini-projects section, more advanced applications such as trading strategies and cross-asset analysis are introduced, bridging the gap between theory and real-world quantitative finance.

TL;DR:
Build strong intuition in financial time series
Implement models from scratch
Analyze real-world financial datasets
Bridge statistical methods → quantitative trading
